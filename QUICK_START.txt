FAQ & TROUBLESHOOTING
===========================================================

Q: Do I need to migrate existing videos?
A: No! Existing videos continue to work. New columns default to 0/null.

Q: Can users edit videos?
A: No, only admins in the admin panel. Good for content control.

Q: What if a channel is deleted?
A: Videos keep their channel_name text. Optional channel_id becomes null.

Q: How do I track video metrics in the app?
A: Use the SQL helper queries in SQL_COMPLETE_QUERIES.txt
   Increment views on play start, track duration, save to engagement table

Q: Can I revert these changes?
A: Yes, keep backup of original database. New columns have defaults.
   Tables can be dropped if needed (save data first).

Q: Will this affect user data?
A: No, only admins data is extended. User records auto-created via trigger.

Q: What about offline videos?
A: Metrics tracking requires backend. Offline videos still work normally.

Q: Is 16:9 aspect ratio required for all videos?
A: No, thumbnails scale to fit 16:9. Any size image will display correctly.
